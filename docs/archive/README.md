# Archive

This directory contains historical docs that are no longer the current source of truth.

Use archive material for:

1. archaeology
2. understanding previous naming or architecture
3. tracing how a current decision evolved

Do not use archive docs as the current contract unless an active doc explicitly sends you there for historical context.

## Structure

1. `reference/` — superseded reference docs
2. `plans/` — superseded plans
3. `milestones/` — historical milestone contracts
4. `learnings/` — older investigations that were not promoted
5. `expansion/` — speculative or future-facing docs that are not current commitments

## Read Current Docs First

Before opening anything in archive, check:

1. [Docs Home](../README.md)
2. [Reference Index](../reference/README.md)
3. [Plans Index](../plans/README.md)
