# Learnings

This directory contains time-bound investigation notes, market research, and decision records that are not yet stable enough to become reference docs.

Use `learnings/` for:

1. market maps
2. comparative research
3. spike notes
4. evaluation writeups
5. decision records that may later graduate into `reference/`

Do not use `learnings/` for durable product contracts. When a learning becomes stable current state, fold it into the relevant doc under `docs/reference/` and leave a short graduation note in the learning if needed.
