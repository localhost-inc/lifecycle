# Security Policy

## Supported Versions

Only the current `main` branch is supported for security fixes.

| Version | Supported |
| ------- | --------- |
| `main`  | yes       |

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Use one of these private paths instead:

1. GitHub Security Advisories for this repository, if available
2. Direct private contact with the repository owner through GitHub

Please include:

1. A clear description of the issue
2. Reproduction steps or a proof of concept
3. Impact assessment
4. Any suggested mitigation or fix, if you have one

I will acknowledge valid reports as quickly as possible and coordinate remediation privately before disclosure.
