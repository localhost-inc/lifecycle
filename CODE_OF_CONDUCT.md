# Code of Conduct

This repository is a professional collaboration space. Participation in issues, pull requests, and review threads requires respectful, direct, and constructive behavior.

## Expected Behavior

1. Be respectful and precise.
2. Critique ideas and implementations, not people.
3. Keep feedback actionable and grounded in the code or product contract.
4. Assume good faith, but do not hand-wave technical disagreements.

## Unacceptable Behavior

1. Harassment, threats, or discriminatory language
2. Personal attacks or repeated bad-faith argument
3. Doxxing, privacy violations, or publication of secrets
4. Spam, trolling, or deliberate disruption of review workflows

## Enforcement

If you need to report a conduct issue, contact the repository owner privately through GitHub. Maintainers may remove comments, close threads, or block participation when behavior crosses these boundaries.
